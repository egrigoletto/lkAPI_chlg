## Descrição

Este APP utilizando o framework NestJS irá conectar a uma base de dados Remota e a partir dela fazer um CRUD de uma base de dados de Produtos.
Através de um endpoint, será possível fazer operações básicas.
Para fazer uma adição deve ser passada para a rota um json com os seguintes dados:

- Tipo: um tipo qualquer,
- Desc: a descrição do produto,
- marca: a marca do mesmo em texto livre,
- preco: um valor numérico com separador de casas "." que identifica o valor do produto

Link para  o Nest - [Nest](https://github.com/nestjs/nest)

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# executa
$ npm run start

# executa no modo de watch
$ npm run start:dev

# modo de produção
$ npm run start:prod
```
